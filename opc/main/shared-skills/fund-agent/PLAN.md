## 三层架构

**替代方案：** 用 mx-data 替代自建 API，东方财富权威数据 > 自己爬。

### 🔍 研究层（Research Layer）— 分析引擎

| 模块 | 作用 | 实现 |
|------|------|------|
| **mx-search** | 新闻/公告/研报扫描 | 持仓异动时自动触发 |
| **🇺🇸 trump-monitor** | 特朗普讲话追踪 → 关税/制裁/政策风险 | mx-search 自定义查询 |

| **factor-score.py** | 基金评分卡 → 成长/稳定/风险 1-10 分 | 基于净值趋势、最大回撤 |

### ⚖️ 决策层（Decision Layer）— 大脑

| 模块 | 作用 | 规则 |
|------|------|------|
| **risk-guardrail** | 风险护栏 → 止损/预警/减仓触发 | 亏10%提醒，15%强制止损 |
| **portfolio-suggestion** | 操作建议 → 增持/减持/观察 | 综合评分 + 风险 + 事件 |
| **report-generator** | 日报/周报生成 | 结构化报告，邮件推送 |

## 🇺🇸 特朗普讲话追踪（重点！）

### 为什么重要
- **关税政策** → 直接影响出口/新能源板块
- **制裁清单** → 影响科技/半导体
- **美联储/美元政策** → 影响黄金/美股
- **能源政策** → 影响石油/光伏

### 实现方案

```python
# scripts/event-impact.py 核心逻辑

# 1. 定时扫描（每2小时）
搜索词：
- "Trump tariff" / "特朗普 关税"
- "Trump sanctions" / "特朗普 制裁"
- "Trump trade war" / "特朗普 贸易战"
- "Trump oil" / "特朗普 石油"
- "Trump gold" / "特朗普 黄金"

# 2. 事件分类
关税 → 利空出口/利好国内替代
制裁 → 利空科技/半导体
贸易战 → 利空A股整体
能源 → 影响石油/新能源

# 3. 持仓关联
遍历 positions.json → 判断哪些基金受影响

# 4. 风险评分
高（直接影响）→ 红色预警
中（间接影响）→ 黄色提醒
低（几乎无影响）→ 记录不通知
```

### 影响映射表

| 事件类型 | 影响板块 | 操作建议 |
|---------|---------|---------|
| 加征关税 | 出口/新能源 | 减仓相关基金 |
| 制裁科技 | 半导体/AI | 观望，不追高 |
| 美元加息 | 黄金/美股 | 黄金持有，美股观望 |
| 能源政策 | 石油/光伏 | 关注波动，不做判断 |
| 贸易谈判 | A股整体 | 减仓观望 |

## 基金评分卡（factor-score）

### 评分维度（1-10分）

| 维度 | 权重 | 数据来源 |
|------|------|---------|
| **收益能力** | 30% | 近1月/3月/6月回报 |
| **稳定性** | 25% | 最大回撤、波动率 |
| **排名** | 20% | 同类排名分位 |
| **规模** | 15% | 基金规模（太小有清盘风险） |
| **基金经理** | 10% | 任期、历史业绩 |

### 总分 → 操作
- 8-10分：✅ 优质，可加仓
- 6-7分：⚠️ 中等，持有观望
- 4-5分：🟡 较差，考虑减仓
- 1-3分：🔴 垃圾，建议清仓

```
  ↓

  ↓
[研究层] factor-score 评分 + event-impact 分析
  ↓
[决策层] risk-guardrail 检查风险线
  ↓
[决策层] portfolio-suggestion 生成建议
  ↓
[决策层] report-generator 输出报告
  ↓
给用户
```
