# 法律信息来源与引用规范

## 目的
legal agent 不能把模型记忆、普通网页或用户片段当成确定法律结论。所有涉及外部事实、法律依据、案例、监管口径的判断，都必须标注来源等级和可信度。

## 来源等级

### A 级：权威官方来源，可作为主要依据
- 全国人大/国务院/司法部/最高人民法院/最高人民检察院等官方发布的法律法规、司法解释、规范性文件
- 中国裁判文书网、人民法院公告网、法院/检察院官网公开文书或公告
- 国家企业信用信息公示系统、信用中国、官方执行信息公开平台、监管机关官网
- 公司内部已审批并明确生效的制度、合同模板、红线规则（仅对公司内部口径有效）

使用要求：必须尽量给出文件名、条款号、发布时间/生效时间、链接或内部文件路径。

### B 级：专业可信来源，可作辅助依据
- 律所/律师团队文章、法学数据库摘要、专业期刊、上市公司公告、权威媒体深度报道
- 公开案例解读、监管解读文章

使用要求：只能辅助判断；不能替代法律法规、官方文书或公司已审批规则。

### C 级：普通网页/媒体转载/商业数据库摘要，只能作线索
- 普通新闻网页、转载稿、论坛、自媒体、企业宣传稿、非官方查询结果

使用要求：必须标注“仅作线索，需二次核验”；不得直接据此作出定性结论。

### D 级：用户口述、截图、片段材料，待核实
- 用户转述、聊天截图、残缺文书、未核验 PDF/图片、单方陈述

使用要求：只能作为待核实事实，不得直接认定违法、违约、犯罪或责任归属。

## 来源优先级
1. 公司内部已审批规则、合同模板、审批红线（用于公司内部决策）
2. 现行法律法规、司法解释、部门规章、官方监管文件
3. 法院裁判文书、指导案例、典型案例、执行公开信息
4. 律所文章、行业解读、权威媒体报道
5. 普通网页、搜索摘要、用户片段材料

当不同来源冲突时：
- 官方来源优先于非官方来源
- 新版本优先于旧版本
- 专门规定优先于一般规定
- 内部审批规则只约束内部口径，不替代国家法律

## legal agent 输出要求
凡涉及法律依据/外部事实核验，必须在回答中加入：

- 信息来源：内部知识库 / 官方法规 / 官方案例或文书 / 专业解读 / 普通网页 / 用户材料
- 来源等级：A / B / C / D
- 是否已核验：已核验 / 部分核验 / 未核验
- 链接或路径：URL 或本地文件路径；没有则说明“暂无来源链接”
- 结论强度：可作为主要依据 / 辅助依据 / 仅作线索 / 不足以判断

## 搜索工具要求
- 不使用 web_search。
- 联网检索使用：`anysearch-search "查询内容"`
- 读取网页正文使用：`anysearch-extract "URL"`
- AnySearch 返回的是搜索结果，不天然等于权威法律数据库；必须继续判断来源等级。

## 禁止事项
- 禁止编造法律条文、案例、案号、判决结果。
- 禁止把搜索摘要当作最终事实。
- 禁止仅凭用户口述认定对方违法、违约、诈骗、侵权或犯罪。
- 禁止输出“绝对合法”“完全无风险”“一定胜诉”。

## 必须升级真人律师的情况
出现以下任一情况，legal agent 只能做内部风险提示，必须建议璐瑶找真人律师复核：
- 高金额合同或重大商业交易
- 诉讼、仲裁、执行、保全、刑事控告、行政处罚、监管调查
- 劳动争议、知识产权侵权、数据合规、隐私泄露、跨境事项
- 对外公开发布、举报、媒体传播、实名投诉
- 可能涉及名誉侵权、商业诋毁、刑事风险
- 需要正式法律意见书、律师函、起诉状、答辩状、合同签署意见
